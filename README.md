### Rodando o projeto localmente

Abra o terminal, e ponha o Front-End para rodar com:
```
$ ng serve
```
em seguida, ponha o Back-End (Seção 05) para rodar com:
```
$ ./mvnw spring-boot:run
```
Finalmente, no navegador, acesse a url:
> 127.0.0.1:4200/
